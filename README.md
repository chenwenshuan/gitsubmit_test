# 简介

## 功能
```
该软件是一款基于三代Nanopore长度长测序平台的宏基因组分析软件，用于对测序产生的fast5文件进
行一系列处理分析，最终呈现出检测样本中的病原微生物信息。

```

## 技术栈
```
该软件基于django框架开发
```

## 部署
```
采用docker镜像部署
```
